# 皮卡堂维基百科

一个专门为皮卡堂游戏打造的维基百科网站。

## 项目结构

```
pikado-wiki/
├── index.html          # 主页
├── styles.css          # 样式文件
├── script.js           # JavaScript功能
├── pages/              # 子页面
│   ├── characters.html # 角色页面
│   ├── items.html      # 道具页面
│   └── guides.html     # 攻略页面
├── images/             # 图片资源
├── data/               # 数据文件
└── README.md           # 项目说明
```

## 功能特性

- 响应式设计，支持移动端
- 角色图鉴系统
- 道具数据库
- 游戏攻略
- 搜索功能
- 现代化UI设计

## 使用方法

1. 直接在浏览器中打开 `index.html`
2. 或者使用本地服务器运行项目
